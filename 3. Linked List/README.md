Complexity comparision between Array and linked list
|                                |     Array      |    Linked List    |
|--------------------------------|----------------|-------------------|
|Indexing                        |      O(1)      |        O(N)       |
|Insert/ Delete Element At Start |      O(N)      |        O(1)       |
|Insert/ Delete Element At End   | O(1)-amortized |        O(N)       |
|Insert/ Delete Element At Middle|      O(N)      |        O(N)       |
